## 🎯 Sobre o Projeto
Este projeto é uma ferramenta de automação desenvolvida em Python para extrair dados (leads) da web de forma eficiente. Utilizando o **Playwright** para navegação e renderização de páginas dinâmicas, e o **Pandas** para estruturação e limpeza dos dados, o script transforma informações brutas da internet em planilhas organizadas e prontas para análise.

## ✨ Funcionalidades
* **Navegação Automatizada:** Interação com elementos da página e contorno de carregamentos dinâmicos via Playwright.
* **Extração de Dados:** Coleta precisa de informações de contato e métricas relevantes (Leads).
* **Estruturação Analítica:** Tratamento e organização dos dados extraídos utilizando DataFrames do Pandas.
* **Exportação Prática:** Geração de arquivos estruturados (como `.xlsx` ou `.csv`) ignorados pelo controle de versão para segurança dos dados.

## 🚀 Tecnologias Utilizadas
* **[Python](https://www.python.org/)**: Linguagem principal.
* **[Playwright](https://playwright.dev/python/)**: Automação de browser para raspagem de dados em páginas dinâmicas.
* **[Pandas](https://pandas.pydata.org/)**: Biblioteca para manipulação e análise de dados.
